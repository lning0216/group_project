Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">link button</a>

Home

In-text link

CSS selector:

.intext_link, .intext_link:hover, .intext_link:focus

Sample code:

<a class="intext_link" href ="www.sfu.ca">in-text link</a>

Rendered element:

in-text link
